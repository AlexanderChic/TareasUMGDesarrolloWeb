📖 Descripción del Curso

Estructuras de Datos es un curso fundamental que se enfoca en el estudio y análisis de diferentes formas de organizar y almacenar datos en la memoria de la computadora. Los estudiantes aprenden a implementar y utilizar estructuras como arrays, listas enlazadas, pilas, colas, árboles y grafos, además de algoritmos de ordenamiento y búsqueda. El curso incluye análisis de complejidad temporal y espacial para evaluar la eficiencia de los algoritmos.

🏗️ Principales Estructuras de Datos

📋 Array (Arreglo)

Uso: Almacenamiento de elementos del mismo tipo en posiciones contiguas

✅ Ventaja: Acceso directo O(1)

❌ Desventaja: Tamaño fijo

🔗 Lista Enlazada

Uso: Colección dinámica de elementos conectados por punteros

✅ Ventaja: Tamaño dinámico

❌ Desventaja: No acceso directo

📚 Pila (Stack)

Uso: Estructura LIFO para llamadas de función, expresiones

✅ Ventaja: Operaciones rápidas

❌ Desventaja: Acceso limitado

🚶‍♂️ Cola (Queue)

Uso: Estructura FIFO para procesamiento secuencial

✅ Ventaja: Orden preservado

❌ Desventaja: Acceso limitado

🌲 Árbol Binario

Uso: Búsquedas eficientes, expresiones jerárquicas

✅ Ventaja: Búsqueda O(log n)

❌ Desventaja: Puede degenerar

#️⃣ Hash Table

Uso: Búsqueda rápida con clave-valor

✅ Ventaja: Acceso O(1) promedio

❌ Desventaja: Colisiones posibles

💾 Implementación de Árbol Binario de Búsqueda

class NodoArbol {
public:
    int dato;
    NodoArbol* izquierdo;
    NodoArbol* derecho;
    
    NodoArbol(int valor) {
        dato = valor;
        izquierdo = nullptr;
        derecho = nullptr;
    }
};

class ArbolBinarioBusqueda {
private:
    NodoArbol* raiz;
    
    NodoArbol* insertar(NodoArbol* nodo, int valor) {
        if (nodo == nullptr) {
            return new NodoArbol(valor);
        }
        
        if (valor < nodo->dato) {
            nodo->izquierdo = insertar(nodo->izquierdo, valor);
        } else if (valor > nodo->dato) {
            nodo->derecho = insertar(nodo->derecho, valor);
        }
        
        return nodo;
    }
    
    bool buscar(NodoArbol* nodo, int valor) {
        if (nodo == nullptr) {
            return false;
        }
        
        if (valor == nodo->dato) {
            return true;
        } else if (valor < nodo->dato) {
            return buscar(nodo->izquierdo, valor);
        } else {
            return buscar(nodo->derecho, valor);
        }
    }
    
public:
    ArbolBinarioBusqueda() { raiz = nullptr; }
    
    void insertar(int valor) {
        raiz = insertar(raiz, valor);
    }
    
    bool buscar(int valor) {
        return buscar(raiz, valor);
    }
};


🔄 Algoritmos de Ordenamiento

🫧 Bubble Sort

Complejidad: O(n²)

Intercambia elementos adyacentes

🎯 Selection Sort

Complejidad: O(n²)

Selecciona el mínimo repetidamente

📝 Insertion Sort

Complejidad: O(n²)

Inserta en posición correcta

🔀 Merge Sort

Complejidad: O(n log n)

Divide y conquista estable

⚡ Quick Sort

Complejidad: O(n log n)

Particionamiento con pivot

🏔️ Heap Sort

Complejidad: O(n log n)

Usa estructura de heap
